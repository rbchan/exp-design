%\vfill
We could compute AIC using the equation
$\mathrm{AIC}=n\log(\mathrm{RSS}/n)+2K$, where RSS is the residual
sum-of-squares.
\pause
\vfill
However, we will use the more general formula: $\mathrm{AIC} =
-2\mathcal{L}(\hat{\theta} ; {\bf y}) + 2K$.
